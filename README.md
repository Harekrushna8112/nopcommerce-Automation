nopCommerce Automation Testing (Selenium + Java)

Project Description

This project contains automated test scripts for the nopCommerce demo website using Selenium WebDriver with Java.
The framework is built using Maven for dependency management.

Technologies Used

- Java
- Selenium WebDriver
- Maven
- TestNG
- ChromeDriver
- Visual Studio Code

Project Structure

src/test/java/com/example → Test cases/ reusable methods
src/main/java/com/example → main method
pom.xml →  dependencies

How to Run the Project

1. Clone the repository
2. Open the project in IDE (VS Code )
3. Install Maven dependencies
4. Run the MainTest.java file 

Website Tested

https://demo.nopcommerce.com/

Test Scenarios

Home page
Login functionality
Resister functionality
Add to cart
Product checkout
